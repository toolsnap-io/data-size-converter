# Data Size Converter 💾

Convert between digital storage units instantly. From bytes to petabytes and everything in between.

## Features

- Convert between bytes, KB, MB, GB, TB, PB, and more
- Binary (KiB, MiB) and decimal (KB, MB) standards
- Real-time conversion as you type
- Bit-level conversions (bits, Kbit, Mbit, Gbit)
- Copy results to clipboard

## Supported Units

- **Bytes:** B, KB, MB, GB, TB, PB, EB
- **Binary:** KiB, MiB, GiB, TiB, PiB
- **Bits:** bit, Kbit, Mbit, Gbit

## Usage Examples

- 1 GB = 1,024 MB = 8,589,934,592 bits
- 1 TB = 931.32 GiB
- 100 Mbps = 12.5 MB/s
